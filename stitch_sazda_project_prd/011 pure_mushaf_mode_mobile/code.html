بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ
الٓمٓ ١ ذَٰلِكَ الْكِتَٰبُ لَا رَيْبَ ۛ فِيهِ ۛ هُدًى لِّلْمُتَّقِينَ ٢ الَّذِينَ يُؤْمِنُونَ بِالْغَيْبِ وَيُقِيمُونَ الصَّلَٰوةَ وَمِمَّا رَزَقْنَٰهُمْ يُنفِقُونَ ٣ وَالَّذِينَ يُؤْمِنُونَ بِمَآ أُنزِلَ إِلَيْكَ وَمَآ أُنزِلَ مِن قَبْلِكَ وَبِالْءَاخِرَةِ هُمْ يُوقِنُونَ ٤ أُو۟لَٰٓئِكَ عَلَىٰ هُدًى مِّن رَّبِّهِمْ ۖ وَأُو۟لَٰٓئِكَ هُمُ الْمُفْلِحُونَ ٥
٢
