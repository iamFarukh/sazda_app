# PRD: Sazda - Modern Islamic App ## Project Overview Sazda is a comprehensive Islamic mobile application designed to be a daily companion for Muslims of all knowledge levels. It balances spiritual growth with practical utility tools. ## Visual Identity - **Name:** Sazda (Focus on "Sujood" / Prostration) - **Tagline:** "Connect. Reflect. Grow." - **Core Palette:** Deep Emerald Green (#064E3B), Soft Beige (#F5F5DC), Gold Accents (#D4AF37), Pure White. - **Typography:** Modern Sans-serif (e.g., Inter/Roboto) for UI + Elegant Arabic Calligraphy (e.g., Amiri/Amiri Quran) for religious texts. - **Style:** Minimalist, spiritual, calming, high accessibility. ## Navigation Structure (Bottom Tab Bar) 1. **Home:** Dashboard, Prayer Times, Daily Inspiration. 2. **Quran:** Surah list, Reader, Audio, Tafsir. 3. **Tools:** Qibla, Tasbeeh (FAB), Zakat, Inheritance, Maps. 4. **Community:** Dua Wall, Knowledge Hub. 5. **Profile:** Settings, Streaks, Achievements. ## Feature List & Screen Requirements ### 1. Auth & Onboarding - Splash Screen (Logo & Tagline) - Onboarding (3 Slides: Prayer, Quran, Community) - Permissions (Location for Prayer Times) - Sign In / Guest Mode ### 2. Home Tab - Dashboard (Greeting, Current Prayer, Today's Verse) - Prayer Time Details (Full day schedule) - Adhan Settings (Muezzin selection) - Daily Verse (Full screen focus) - Mood Guidance (Islamic advice based on mood) - Sunnah Tracker (Habit building) ### 3. Quran Tab - Quran Home (Continue reading, Popular Surahs) - Surah List (114 Surahs, Search) - Quran Reader (Clear Arabic text, word-by-word) - Audio Player (Reciter selection, loop) - Bookmarks & Last Read - Tafsir View ### 4. Tools Tab - Tools Dashboard - Qibla Finder (Compass + AR) - Tasbeeh Counter (Haptic feedback, goals) - Zakat Calculator (Multi-asset input) - Inheritance Calculator (Family tree/logic) - Mosque & Halal Map - Hijri Calendar - Hajj & Umrah Guide ### 5. Community Tab - Dua Wall (Feed with "Ameen" interaction) - Post Dua (Anonymous option) - Knowledge Hub (Ask Imam, Categories) - Fatwa Details ### 6. Profile Tab - Profile Dashboard (Streaks, Badges) - General Settings - Notification Settings - Theme/Appearance (Light/Dark toggle) - Offline Downloads
